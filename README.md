# Godot-SRPG-DEMO

一个基于Godot引擎的回合制SRPG游戏演示项目，展示了回合制战斗、网格地图、敌人AI等核心SRPG元素。

## 功能特性

- **回合制战斗系统**：基于速度属性的行动顺序，支持轮数计数
- **职业系统**：包含近战、远程和精英三种敌人职业
- **运动轨迹**：所有角色移动时显示路径动画，对角移动拆分为两段直线
- **攻击范围**：显示攻击范围动画，角色中心位置不涂色
- **地图系统**：棋盘式网格布局，清晰的格子显示和障碍物系统
- **AI系统**：敌人根据血量调整攻击欲望，优先攻击玩家
- **精英敌人AI**：精英敌人具备近战和远程攻击能力，根据血量和距离智能选择攻击方式

## 技术栈

- Godot 4.6.1
- C#
- .NET 8.0

## 如何运行

1. 确保安装了Godot 4.6.1或更高版本
2. 打开Godot引擎
3. 导入项目文件夹
4. 点击运行按钮启动游戏

## 游戏操作

- 选择玩家角色：点击玩家单位
- 移动：点击高亮的格子
- 攻击：选择攻击类型后点击范围内的敌人
- 结束回合：点击"结束回合"按钮

## 项目结构

- `src/`：游戏源代码
  - `GameManager.cs`：游戏主逻辑，包括地图绘制、单位管理、AI逻辑等
  - `Unit.cs`：单位属性和行为
  - `TurnManager.cs`：回合管理
  - `Grid.cs`：地图网格系统

## 敌人AI行为

- **普通敌人**：根据血量和距离决定攻击或移动
- **精英敌人**：
  - 距离玩家1格且血量≥50%：使用近战攻击
  - 距离玩家1格且血量<50%：尝试撤退到远程位置后使用远程攻击
  - 距离玩家>1格：根据距离和血量决定攻击或移动
